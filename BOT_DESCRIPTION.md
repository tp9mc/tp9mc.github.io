# PropFerma Bot — описание функционала

## Общее

Telegram-бот (@PropFerma) + Telegram Mini App (https://tp9mc.github.io).
Пользователь открывает конструктор интерьера, выбирает параметры, нажимает кнопку —
бот генерирует фотореалистичный рендер комнаты и отправляет его с детальным отчётом.

---

## Команды бота

| Команда    | Что делает                                                     |
|------------|----------------------------------------------------------------|
| `/start`   | Приветствие + кнопка «🏠 Открыть конструктор»                  |
| `/restart` | То же самое (перезапуск)                                       |
| `/app`     | Показывает кнопку открытия Mini App                            |
| `/stats`   | Отправляет txt-файл со статистикой (всё о всех пользователях)  |

## Mini App — конструктор интерьера

### Шаг 1: Стиль интерьера (3 варианта)
- **Japandi** — тёплое дерево, лён, низкие силуэты, вабисаби
- **Modern Classic** — мрамор, бархат, симметрия, латунь
- **Scandinavian** — светлая берёза, белые стены, шерстяной текстиль

### Шаг 2: Комната (4 варианта)
- Гостиная / Спальня / Ванная / Кухня

### Шаг 3: Конфигурация (27 слотов)
Три категории по 9 слотов каждая:

| Категория    | Что настраивается                              |
|--------------|------------------------------------------------|
| Мебель       | 9 предметов мебели для выбранной комнаты       |
| Освещение    | 9 типов световых решений                       |
| Материалы    | 9 отделочных материалов и деталей              |

В каждом слоте 2 варианта (A / Б). Итого до 27 выборов на генерацию.
Варианты зависят от стиля и комнаты — всего 3×4 = 12 уникальных наборов.

### Кнопка «Сгенерировать»
При нажатии Mini App отправляет все выборы боту через `web_app_data`.

## Генерация изображения

### Основной путь — HuggingFace FLUX.1-schnell
- Модель: `black-forest-labs/FLUX.1-schnell` через `router.huggingface.co`
- Размер: 1344×768 px
- Таймаут: 120 секунд
- Стоимость: **$0.0017 за генерацию** (измерено: $1.40 / 802 запроса, апрель 2026)

### Резервный путь — локальный SD WebUI (Juggernaut XL)
Активируется если HF недоступен или вернул ошибку.
- SD WebUI запускается автоматически если не запущен (ждёт до 120 сек)
- Параметры: Euler / Karras, 30 шагов, CFG 10, 1344×768
- Стоимость: $0.00 (локальная генерация)
- Watchdog: SD WebUI выключается автоматически через **15 минут** простоя

### Промт
Собирается динамически из:
1. DNA стиля (ключевые материалы и атмосфера)
2. DNA комнаты (тип пространства)
3. Английские ключевые фразы из каталога для каждого выбранного предмета
4. Финальные теги: `professional interior photography, natural daylight, wide angle lens, 8k, architectural digest`

Негативный промт: базовый список (`people, ugly, blurry, watermark...`) + негативы из каталога.

## Ответ пользователю

После генерации бот отправляет два сообщения:

1. **Фото** с подписью: `✅ Готово за Xс  •  Стиль · Комната`

2. **Текстовый файл** `report_style_room.txt` с полной конфигурацией:
   - Дата и время
   - Стиль и комната
   - Все 27 слотов с выбранными вариантами и названиями предметов
   - Технические параметры генерации:
     - HF: модель, провайдер, размер, время, стоимость, промт
     - SD: модель, сэмплер, шаги, CFG, seed, стоимость, промт + негативный промт

## Режим редактирования сайта (Edit Mode)

Доступен через серую точку в правом верхнем углу Mini App.

**Текстовые блоки** — все надписи на сайте редактируются in-place (сохраняются в localStorage).

**Картинки** — у каждого изображения появляется точка. При нажатии открывается модал:
- Предпросмотр текущего изображения
- Загрузка новой картинки с устройства
- Промт и негативный промт (предзаполнены из оригинальных параметров генерации)
- **Кнопка «⚡ Сгенерировать по промту»** — вызывает HF FLUX.1-schnell прямо из браузера:
  - Первый раз запрашивает HF API Token (сохраняется в localStorage)
  - Лимит: **100 платных генераций в час** (защита от случайного перерасхода)
  - Блокирующий оверлей со счётчиком секунд на время генерации
  - После генерации картинка сохраняется и сайт обновляется автоматически
- Кнопка «Сбросить к оригиналу» — удаляет все правки для этого изображения

Предзаполненные промты: все 663 изображения сайта (3 стиля + 12 комнат + 648 иконок опций)
имеют оригинальные промты. Иконки подгружаются лениво из `prompts.json` при первом
включении режима редактирования.

## Статистика (`/stats`)

Отчёт формируется из лога событий (`/tmp/bot_stats.jsonl`).

**Глобально:**
- Всего событий и уникальных пользователей
- Успешных / неуспешных генераций с разбивкой HF vs SD
- Среднее время генерации
- Суммарные расходы HF (бот)

**По стилям и комнатам:** топ по числу генераций

**По каждому пользователю:**
- Username и user_id
- Всего действий, генераций (HF / SD), расходы HF
- Дата первого и последнего визита
- Детализация всех типов событий

**Полный лог** всех событий с таймстемпами.

## Ежедневный бэкап (03:00 через launchd)

Скрипт `backup.sh` запускается macOS launchd каждую ночь в 03:00:
1. Git commit + push незакоммиченных изменений на GitHub
2. Архив кода репозитория (`code-YYYY-MM-DD.tar.gz`)
3. Копии файлов данных: `/tmp/bot_stats.jsonl`, `/tmp/catalog.json`
4. Ротация: хранятся только последние 14 дней
5. Лог: `~/claude-workspace/backups/tp9mc/backup.log`

## Технический стек

| Компонент       | Технология                                  |
|-----------------|---------------------------------------------|
| Бот             | Python, pyTeleBot                           |
| Mini App        | Vanilla JS, HTML/CSS (один файл index.html) |
| Хостинг сайта   | GitHub Pages (tp9mc.github.io)              |
| Генерация (primary) | HuggingFace FLUX.1-schnell              |
| Генерация (fallback) | Stable Diffusion WebUI (Juggernaut XL) |
| Каталог предметов | `/tmp/catalog.json` (291 KB, 648 предметов) |
| Промты иконок   | `prompts.json` (220 KB, 648 записей)        |
| Хранение правок | localStorage браузера                       |
| Статистика      | JSONL-лог `/tmp/bot_stats.jsonl`            |
| Бэкап           | launchd + git + tar.gz                      |

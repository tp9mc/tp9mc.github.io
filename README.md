# Кабина пилота — Lamoda Tech: Parsing × Matching

Live: **https://tp9mc.github.io**

Продуктовый стенд Head of Product: два полностью рабочих флагманских
инструмента, работающих в автоматическом режиме, и дашборд с объективной
статистикой по всем их метрикам.

## Что внутри

| Компонент | Что делает |
|---|---|
| **LT Parsing** (`tools/parsing/`) | Краулер конкурентных площадок: настоящий HTTP-обход трёх источников разного типа (HTML+sitemap, HTML+пагинация, JSON API), извлечение офферов, ретраи, телеметрия каждого запроса |
| **LT Matching** (`tools/matching/`) | Движок сопоставления офферов с собственным каталогом: IDF-скоринг, блокинг, авто-пороги, очередь ручной валидации, ценовой индекс. Precision/recall считаются честно против голдсета |
| **Симулятор рынка** (`tools/market_sim/`) | Три демо-магазина (`shops/`) с живым ассортиментом: цены дрейфуют, товары появляются/уходят, страницы ломаются, магазины уходят в maintenance, попадаются капчи |
| **Кросс-аналитика** (`tools/analytics/`) | E2E метрики пайплайна, алерты деградаций, сводка для дашборда |
| **Кабина пилота** (`index.html`, `assets/`) | Дашборд: KPI, тренды, воронки, таблицы, журнал инцидентов. Тёмная/светлая тема, у каждого графика — табличный вид |
| **PO-боты** (`tools/po_bots/`) | Лера (PO Parsing) и Марк (PO Matching): дайджесты, алерты и ответы на вопросы в групповом Telegram-чате |

## Автоматический режим

GitHub Actions запускает полный цикл **каждые 3 часа** (`.github/workflows/pipeline.yml`):

```
симуляция рынка → LT Parsing (HTTP-обход) → LT Matching → аналитика
      → коммит data/ и shops/ в main → пересборка Pages → отчёты ботов
```

Раз в 15 минут (`po-bots.yml`) боты забирают вопросы из Telegram и отвечают.

## Локальный запуск

```bash
pip install -r tools/requirements.txt
python -m tools.run_pipeline                 # один прогон
python -m tools.run_pipeline --backfill-days 7 --runs-per-day 4  # история
python -m http.server 8000                   # открыть http://localhost:8000
```

## Подключение Telegram-ботов

1. В [@BotFather](https://t.me/BotFather): `/newbot` дважды — например,
   `LT Parsing PO (Лера)` и `LT Matching PO (Марк)`. Сохраните оба токена.
2. Там же для каждого бота: `/setprivacy` → **Disable** (чтобы боты видели
   обычные сообщения в группе, а не только команды).
3. Создайте групповой чат, добавьте обоих ботов.
4. В настройках репозитория → Secrets and variables → Actions добавьте:
   `PARSING_BOT_TOKEN`, `MATCHING_BOT_TOKEN`.
5. Напишите в группу любое сообщение (например, `/start`) — в течение
   15 минут боты запомнят чат и поздороваются. Дальше всё само:
   дайджесты раз в день, алерты после прогонов, ответы на вопросы.

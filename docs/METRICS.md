# Каталог продуктовых метрик

Все метрики считаются пайплайном на каждом прогоне и доступны на дашборде
(вкладки соответствуют разделам ниже). Ничего не рисуется руками — источник
каждой цифры указан.

## LT Parsing — собственные метрики

### Покрытие (coverage)
| Метрика | Формула / источник | Цель |
|---|---|---|
| Офферы за прогон | число успешно распарсенных карточек | рост |
| Активные источники | источники с непустой выгрузкой без maintenance | 3/3 |
| Покрытие категорий | уникальные категории в выгрузке | — |
| Найдено (discovery) | URL из sitemap / пагинации / API-индекса | — |

### Надёжность (reliability)
| Метрика | Формула | Цель |
|---|---|---|
| Crawl success rate | 1 − (HTTP-ошибки + парс-ошибки + капчи) / запросы | ≥ 97% |
| Uptime прогонов | 1 − прогоны с недоступным источником / все прогоны | ≥ 95% |
| HTTP-ошибки, капчи, ретраи | счётчики телеметрии Fetcher | ↓ |

### Свежесть (freshness)
| Метрика | Формула | Цель |
|---|---|---|
| Возраст данных источника | now − последняя успешная обкачка | ≤ 6 ч |
| Свежесть в SLA | доля источников с возрастом ≤ 6 ч | 100% |

### Качество данных (quality)
| Метрика | Формула | Цель |
|---|---|---|
| Полнота полей | доля офферов с заполненным полем (цена, бренд, категория, цвет, наличие) | цена ≥ 95% |
| Validation error rate | карточки без обязательных полей / офферы | ↓ |

### Производительность (performance)
| Метрика | Формула |
|---|---|
| Страниц в минуту | запросы / длительность |
| Задержка avg / p95 | телеметрия каждого HTTP-запроса |
| Длительность прогона | конец − старт |

### Сигналы рынка (delta)
| Метрика | Формула |
|---|---|
| Новые / исчезнувшие офферы | сравнение ключей с прошлым снапшотом |
| Изменения цен | офферы с изменившейся ценой |
| Доля out-of-stock | офферы «нет в наличии» / все |

## LT Matching — собственные метрики

### Воронка (funnel)
| Метрика | Формула | Цель |
|---|---|---|
| Auto-match rate | авто-мэтчи / офферы на входе | ≥ 60% |
| Итоговый match rate | (авто + подтверждённые валидацией) / офферы | ≥ 85% |
| Кандидатов, пар оценено | телеметрия блокинга и скоринга | — |

### Качество (quality) — против голдсета (ground truth симуляции)
| Метрика | Формула | Цель |
|---|---|---|
| Precision (авто) | верные авто-мэтчи / все авто-мэтчи | ≥ 95% |
| Recall | найденные верные мэтчи / все сопоставимые офферы | ≥ 80% |
| F1 | гармоническое среднее P и R | — |
| False match rate | неверные мэтчи в итоговом наборе / итоговый набор | ↓ |
| Точность валидаторов | верные вердикты / разобранные карточки | ≥ 98% |

### Очередь валидации (review)
| Метрика | Формула | Цель |
|---|---|---|
| Размер очереди | карточки в зоне 0.50–0.74 без вердикта | ≤ 150 |
| Разобрано за прогон | вердикты валидаторов (мощность 70) | — |
| Прогноз разбора | очередь / мощность, в прогонах | ≤ 2 |

### Уверенность и консолидация
| Метрика | Источник |
|---|---|
| Распределение скора | гистограмма лучшего кандидата, 0.50–1.00 |
| SKU с несколькими офферами | консолидация офферов разных магазинов |

### Производительность
| Метрика | Формула |
|---|---|
| Пар в секунду | пары / длительность |
| Пороги | auto 0.74 (+ отрыв 0.04), review 0.50 |

## Пересекающиеся метрики (оба продукта)

| Метрика | Формула | Цель |
|---|---|---|
| **Покрытие ценового индекса (NSM)** | SKU каталога с ≥ 1 сопоставленным оффером и ценой / каталог | рост |
| Ценовая конкурентоспособность | SKU, где наша цена ≤ min конкурентов / сопоставимые SKU | ≥ 50% |
| Средний ценовой разрыв | mean((наша − min конкурентов) / min) | → 0 |
| Покрытие рынка | распарсенные офферы / реально существующие позиции (по ground truth) | ≥ 95% |
| E2E длительность | парсинг + мэтчинг | ≤ 5 мин (SLA) |
| Свежесть end-to-end | свежесть парсинга × периодичность прогонов | ≤ 6 ч |
| Алерты | правила деградаций (источник упал, precision ↓, очередь ↑, покрытие ↓) | 0 критичных |

Качество мэтчинга напрямую зависит от качества парсинга (полнота бренда,
цены, категории) — поэтому полнота полей LT Parsing и precision/recall
LT Matching смотрятся на дашборде рядом, а деградация покрытия индекса
алертится как общий (shared) инцидент.
